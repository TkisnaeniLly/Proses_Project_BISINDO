# Repository Proses Pengembangan Sistem Translasi BISINDO

Repository ini berisi proses pengembangan sistem translasi Bahasa Isyarat Indonesia (BISINDO) untuk tugas mata kuliah Pengolahan Citra Digital.

Perlu diperhatikan bahwa repository ini **tidak sepenuhnya berisi hasil akhir yang bersih**, melainkan kumpulan proses eksplorasi, eksperimen, dan beberapa metode yang digunakan selama pengembangan.  
Beberapa tahapan mungkin tidak terdokumentasi secara lengkap atau dilewati, karena fokus utama repo ini adalah **sharing kode dan alur pengerjaan**.

Di dalam repository ini terdapat beberapa pendekatan dalam melatih model, namun **hasil akhir yang digunakan hanya merupakan salah satu dari metode tersebut**.
