📘 Tutorial Penggunaan

Pilih metode translasi sesuai kebutuhan ✨

🎥 Tutorial Translate Real-Time

  1. Pilih menu Translate → Real-Time
  2. Izinkan akses kamera
  3. Arahkan tangan ke kamera
  4. Tunggu hasil huruf muncul

🖼️ Tutorial Upload Gambar

  1. Pilih menu Translate → Upload Gambar
  2. Klik area upload gambar
  3. Pilih gambar bahasa isyarat
  4. Klik tombol Deteksi
  5. Lihat hasil huruf dan akurasi
